# ADR-0001：使用模块化单体

- 状态：接受
- 日期：2026-07-12

## 决策

Web 前端和一个 FastAPI 服务组成首版系统。FastAPI 内部按路由、Schema、服务、仓储和模型分层；ASR、LLM 与 Embedding 仅作为适配器，不独立部署为业务微服务。

## 原因

个人暑期项目需要优先保证可测试的端到端闭环。模块化单体可共享事务边界，减少部署和可观测性负担，同时保留未来抽离模型推理进程的接口。

## 后果

- 模型加载必须使用应用生命周期单例，避免请求级重复加载。
- CPU/GPU 密集推理需要并发上限，未来可把适配器移到独立进程。
- 数据库迁移和 API 版本仍需保持向后兼容。
